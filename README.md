# You Don't Need jQuery

A beautiful React application demonstrating vanilla JavaScript equivalents for common jQuery patterns.

## Features

- Interactive code comparison between jQuery and vanilla JavaScript
- 8 common jQuery patterns with their vanilla JS equivalents
- Live demo functionality
- Beautiful gradient UI with Tailwind CSS
- Responsive design

## Getting Started

### Install Dependencies

```bash
npm install
```

### Run Development Server

```bash
npm run dev
```

The app will be available at `http://localhost:5173`

### Build for Production

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (icons)
